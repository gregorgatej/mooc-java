# MOOC Java

Contains some of my solutions to exercises from [University of Helsinki’s massive open online course (MOOC)
on programming in Java](https://java-programming.mooc.fi/).


## Requirements

- Java 25 (OpenJDK)

If using SDKMAN:

```bash
sdk install java 25-open
sdk env
```
